## Демо

**[Посмотреть демо на GitHub Pages](https://lina-whm.github.io/portfolio/)**

## Технологический стек

### Frontend
- **React 18** - библиотека для построения пользовательских интерфейсов
- **JavaScript ES6+** - основной язык разработки
- **TypeScript** - статическая типизация для надежности кода
- **CSS3** - стилизация компонентов
- **Framer Motion** - создание плавных анимаций

### Управление состоянием
- **React Hooks** - управление состоянием компонентов

### Роутинг
- **HashRouter** - навигация по страницам портфолио

### UI компоненты
- **React Icons** - иконки для интерфейса
- **Кастомные CSS компоненты** - стилизованные карточки проектов, кнопки, формы

### Оптимизация
- **Lazy Loading** - ленивая загрузка изображений
- **Code Splitting** - разделение кода на чанки
- **Memoization** - оптимизация рендеринга компонентов

### PWA
- **Web App Manifest** - для установки на устройства
- **Responsive Design** - полная адаптация под все устройства

## Типизация TypeScript
- Интерфейсы для пропсов компонентов
- Типизация состояний и событий
- Кастомные типы для проектов и навыков

## Быстрый старт

### Предварительные требования
- Node.js 16 или выше
- npm или yarn

### Установка и запуск
```bash
# Клонирование репозитория
git clone https://github.com/lina-whm/portfolio.git
cd portfolio

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start

# Приложение будет доступно по адресу: http://localhost:3000

# Сборка для production
npm run build

# Деплой на GitHub Pages
npm run deploy
```
