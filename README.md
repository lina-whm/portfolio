# Портфолио Frontend разработчика

Мое персональное портфолио, созданное с использованием React. Здесь представлены мои проекты, навыки и контактная информация.

##  Технологии

- **React** — библиотека для пользовательских интерфейсов
- **Framer Motion** — анимации
- **React Icons** — иконки
- **CSS Modules** — стилизация компонентов
- **GitHub Pages** — хостинг


##  Особенности

- Адаптивный дизайн (от десктопа до мобильных)
- Плавные анимации при скролле
- Фильтрация проектов по технологиям
- Форма обратной связи
- Современный минималистичный дизайн
